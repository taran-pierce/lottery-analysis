# Colorado Powerball Scraper

A Python script to scrape historical **Colorado Powerball** and **Double Play** lottery numbers directly from the Colorado Lottery website. The script can scrape multiple pages, print the results in the terminal, and save them to a CSV file.  

It includes CLI arguments, a progress bar, and automatic CSV file naming.

---

## Features

- Scrapes **Powerball** and **Double Play** numbers
- Supports scraping **multiple pages** of historical results
- Prints results cleanly to the terminal
- Saves results to CSV with **automatic date + pages naming**
- Progress bar for scraping multiple pages
